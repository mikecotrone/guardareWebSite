l'architettura — the architecture

Four microservices. One platform.

Independent mini-apps working together over a centralized database, provisioned entirely through GitHub-driven YAML. Data flows one direction.

raccogliere — to collect

vCollector · Flexible data collection

Around-the-clock endpoint collection over API, SNMP, and Remote PowerShell. Flexible YAML configuration delivers cross-manufacturer support that stays fast, efficient, and tunable.

normalizzare — to normalize

Normalizer · Multi-vendor normalization

Schema-on-Config — ingests API, SNMP, Remote PowerShell, and CLI results into common schemas, with 100% dynamic database creation and updates driven entirely by admin configuration. Define a mapping in YAML and the schema builds itself; revise that YAML later and it updates itself the same way — automatically, at setup and on every change after. No migrations, no hardcoded tables, no code changes.

avvisare — to alert

Events Director · Events & alerting

An intuitive rule-based decision engine with flexible YAML rulesets and Crescendo, the hierarchical escalation engine that silences white noise and stays quiet until something truly matters. It feeds two channels: event and auto-clear notifications to incident-response platforms like PagerDuty, and direct ITSM ticketing with Jitbit and TeamDynamix, where tickets open, update, and resolve automatically under admin-selectable, bi-directional flow control.

documentare — to document

HuduSync · Documentation & asset sync

Hudu is the documentation and asset-management platform many MSPs run their operations on. HuduSync keeps it effortlessly current: publishing every monitored device into the right Hudu asset layouts automatically, with zero duplicates, full lifecycle management, and onboarding that's a YAML edit, never a code change.

i canti — the chapters

Your stack, already scored.

A Canto is a pre-tuned playbook written for specific services and infrastructure devices. Each Canto defines what vCollector gathers, how the Normalizer maps it, and which Events Director escalates — one YAML drop, proven in production before it ships. Anything with an API is within reach; the gear you run every day is ready on day one. And when stock tuning isn't enough, your deployment gets a Cadenza: custom passages, written for your environment.

Request a Canto

Cantos ship with the platform. The community decides what's next.

le funzionalità — the features

Everything observability needs. Nothing it doesn't.

Collection

API, JSON-RPC, RESTConf, SNMP, Remote PowerShell, and Linux libc collection methods.

Normalization

Multi-vendor normalization with user-configurable mapping and a scripting language for data manipulation. Schema-on-Config creates and updates database structures 100% dynamically from admin configuration — no migrations, no hardcoded tables.

Events & alerting

Event triggering with easy rulesets and native incident-response integrations such as PagerDuty and UptimeRobot, plus direct, bi-directional ticketing with Jitbit and TeamDynamix.

Reporting

Intuitive reporting that turns observability data into clear answers: service health, KPIs, and assessment insights, with much more on the way.

Serviceability

Auto-upgrading microservices with GitHub configuration control. Every config is YAML.

Security

Hardened Ubuntu on a pre-configured on-premise microserver. REST over TLS 1.3, normalized data in a localized MariaDB database.

la strada — the road ahead

On the roadmap.

Assessment reporting

Scheduled KPI health reports for Network Operations and Cloud System Operations, with C-level stoplight ratings, systems health, chronic-issue analysis from ITSM tickets, and intrusion-prevention reporting.

Administrator web portal · 2027

Systems monitoring and UI-based configuration management, hosted locally or in the cloud, built on a REST and WebSocket API model. Its completion opens the platform’s source to everyone.

Multi-lingual by design

Guardare carries an Italian name for a reason. The Admin Web UI, reporting, and alert content arrive localized: your platform, in your language.

Container deployment

The full platform — vCollector, Normalizer, and Events Director — as a single Docker Compose stack: one command, on a microserver or in your enterprise virtual environment.
